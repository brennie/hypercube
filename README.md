# Hypercube

A Minecraft HTTP API plugin for servers running [Sponge][sponge].

## Building Hypercube

Hypercube is built with a Docker container:

```sh
docker-compose build builder
docker-compose up builder
```


[sponge]: https://www.spongepowered.org/
